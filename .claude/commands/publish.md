다음을 순서대로 실행하세요:

1. `git status` 로 변경사항 확인.
2. 변경 사항을 한 문장으로 요약하여 commit 메시지 자동 생성 (`feat: ...` 또는 `chore: ...`).
3. `git add .` → `git commit -m "<메시지>"` → `git push`
4. 사용자에게 "올렸습니다! 1-2분 후 https://ta0110.github.io/ 에서 보입니다." 안내.
5. 만약 commit 할 변경사항이 없으면 사용자에게 알리고 종료.
